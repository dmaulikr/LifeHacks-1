# How to Unix

## 0. Linux Update Programs

### Fetch the list of available updates
`sudo apt-get update`

### Strictly upgrades the current packages
`sudo apt-get upgrade`

### Installs updates (new ones)
`sudo apt-get dist-upgrade`
